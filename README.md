# Skills

Repositório de skills para agentes de IA (OpenCode, Claude Code, Codex, Gemini CLI, GitHub Copilot e outros).

## Instalação

Instale a skill desejada com:

```bash
npx skills add Nomadiction8991/Skills --skill <nome-da-skill>
```

## Skills disponíveis

| Skill | Descrição |
|-------|-----------|
| `chamado` | Abre, edita, lê e comenta chamados de suporte no TomTicket |

> Use `npx skills add Nomadiction8991/Skills --list` para ver todas as skills disponíveis no repositório.

## Atualização

Para atualizar todas as skills instaladas para a versão mais recente:

```bash
npx skills update
```

Para verificar se há atualizações disponíveis antes de atualizar:

```bash
npx skills check
```

Para reinstalar uma skill específica com a versão mais recente do repositório:

```bash
npx skills add Nomadiction8991/Skills --skill <nome-da-skill>
```
