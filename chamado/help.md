# /chamado

```
/chamado abrir <descrição do problema>
/chamado editar <número> <o que editar>
/chamado ler <número>
/chamado comentar <número> <mensagem>
```

**Exemplos:**
```
/chamado abrir Erro 500 ao finalizar pedido no módulo de entregas
/chamado editar 3790 ajustar formatação visual do corpo
/chamado editar 3790 adicionar os passos para reproduzir
/chamado ler 3790
/chamado comentar 3790 Cliente informou que o problema persiste após a atualização
```
